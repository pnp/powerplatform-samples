## Minimal Path to Awesome
* [Download](./solution/convert-sharepoint-library-doc-file-to-pdf.zip) the `.zip` from the `solution` folder
* Within **Power Automate**, import the solution `.zip` file using **Solutions** > **Import Solution** and select the `.zip` file you just packed.

## Features

This sample demonstrates the following concepts:

*   Get file content when a file is created to document library
*   Create an actual file to Onedrive
*   Convert a file to PDF 
*   Create a PDF file to Onedrive
*   Create a newly generated PDF file to document library
*   Delete actual and a PDF file from Onedrive
